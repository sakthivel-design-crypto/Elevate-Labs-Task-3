# Elevate Labs - AI & ML Internship

## Task 3: Linear Regression

### Objective

The objective of this task is to implement and understand Linear Regression using Scikit-learn to predict house prices based on house area.

---

## Tools Used

- Python
- Visual Studio Code
- Pandas
- Matplotlib
- Scikit-learn

---

## Dataset

- Housing.csv

---

## Steps Performed

1. Imported the required Python libraries.
2. Loaded the Housing dataset using Pandas.
3. Selected **area** as the input feature and **price** as the target variable.
4. Split the dataset into training (80%) and testing (20%) sets.
5. Trained a Linear Regression model using Scikit-learn.
6. Predicted house prices on the test dataset.
7. Evaluated the model using:
   - Mean Absolute Error (MAE)
   - Mean Squared Error (MSE)
   - R² Score
8. Plotted the regression line to visualize the relationship between area and price.
9. Displayed the slope and intercept of the regression model.

---

## Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

These metrics help measure the prediction accuracy of the Linear Regression model.
